Usage:


Install bower component.

bower install angular-slick --save

Add jquery, angular, slick-carousel and angular-slick.


Add a dependency to the "slick" module.

angular.module('MyApp', ['slick']);

Add a `slick` element to your html.

That's it!

Information about all settings you can find at original slick site.
